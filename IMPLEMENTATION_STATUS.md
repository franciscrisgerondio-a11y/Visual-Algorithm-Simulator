1. Open `index.html` in a web browser
2. Select an algorithm category from the dropdown
3. Choose a specific algorithm
4. Click "Start" to begin the visualization
5. Use Pause/Resume to control the animation
6. Adjust speed slider to change animation speed
7. View code snippets and complexity information
